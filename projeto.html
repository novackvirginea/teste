PROJETO ESCRITRAS BRASILEIRAS

Um projeto com o objetivo de criar uma Base de Dados de escritoras brasileiras dos últimos 200 anos!

2014

O começo

Esse projeto surge de forma simples com a construção de uma lista de escritoras brasileiras como uma reação a frase proferida inumeras vezes tanto em sala de aula quanto por importantes e renomados críticos literários: "Naquele tempo as mulheres não escreviam..."

2018

Quantas mulheres cabem no conceito de escritora?

Se o conceito de escritor, historicamente exclui mulheres, o conceito de escritoras igualmente tende a ser branco, classe média, heterocentrado e cisnormativo. Esse foi o momento da pesquisa de esclarecer os diversos feminismos e buscar alternativas de caolização que lutem contra o apagamento.

2020

A ideia toma forma e vira um projeto de doutorado

O objetivo nesse momento desenvolvido no PPGL/PUCRS (Brasil) consisitiu em mapear a tradicional crítica literária e a história da literatura brasileiras no que diz respeito a produção de mulheres constrastando-as com a lista iniciada em 2014 somanda às reflexões feministas.

2022-2023

Análise e divulgação de uma ampla quantidade de informações: As Humanidades Digitais

Durante o período de 6 meses em formato Doutorado Sanduiche, na Universtity of Miami (EUA), sob supervisão da PhD Linday Thomas, o projeto delimita-se enquanto um projeto de Data Analysis e, por esse motivo, busca contruir uma base de dados das escritoras brasieiras.
